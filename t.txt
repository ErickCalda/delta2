✔ 2. (RECOMENDADO) Crear entorno virtual

Esto evita errores raros:

python -m venv manim_env

Activarlo:

CMD:
manim_env\Scripts\activate
PowerShell:
manim_env\Scripts\Activate.ps1
✔ 3. Instalar Manim

Ahora sí:

pip install manim

Forma fácil:
winget install Gyan.FFmpeg

Luego verifica:

ffmpeg -version
